POS C# Windows Forms App

This POS System is a fully functional restaurant management software built with C# Windows Forms and MySQL database integration using XAMPP. It is designed to streamline restaurant operations, manage orders, track inventory, handle suppliers, and oversee staff roles efficiently.

Key Features:
1. Order Management System:

  • Allows users to select customer orders from multiple categories: Food, Drinks, and Desserts.
  • Price counter to track the total bill and item quantities.
  • Table Management System: Users can assign orders to tables. When an order is placed, the table is marked as occupied, preventing new orders until checkout or void.

2. Storage & Inventory Management:
  • Tracks restaurant inventory for both ingredients (recipes) and utilities (plates, cups, etc.).
  • Low Stock Alerts: Notifies when an item is running low to prevent shortages.

3. Supplier Management:
  • Keeps track of restaurant suppliers and their provided categories.
  • Users can place orders for required supplies and choose the supplier.
  • Orders are logged with date and time for accurate record-keeping.

4. Staff Management System:
  • Add, edit, or remove staff members with detailed information.
  • Staff roles include: Chef, Sous Chef, Waiter, Captain Waiter, and Manager Assistant.
  • Maintains attendance records to track workdays and absences.

5. Manager Dashboard & Sales Analytics:
  • Provides detailed sales reports to track revenue.
  • Best-selling product analysis to identify the most popular items.
  • Rating System: Helps managers evaluate product performance.
  • Staff Attendance Records: Managers can review staff workdays and absences.

Technology Used: C# (Windows Forms), MySQL (XAMPP), Windows Forms UI.

This POS System is a complete solution for restaurant management, helping businesses efficiently handle orders, manage inventory, track suppliers, oversee staff, and monitor sales performance. It ensures smoother operations, improves decision-making, and enhances productivity.

Chess Game (Web)

The Chess Game is a fully functional web-based chess game that allows users to play chess against another player. Built with HTML, CSS, JavaScript, and jQuery, this game provides a smooth user experience with intuitive gameplay.

Key Features:

  • Multiplayer Gameplay: Two players can engage in a classic chess game directly through the browser.

Technology Used: HTML, CSS, JavaScript, jQuery.

Fitness Tracking App(Web)

The Fitness Tracking App is a comprehensive web application designed to help users monitor their daily activities, track their diet, and plan their workouts effectively. Built with a user-friendly interface, it aims to provide a seamless experience for anyone looking to achieve their fitness goals.

Key Features:

  • BMI and BMR Calculator: Users can input their weight, height, age, and gender to calculate their Body Mass Index (BMI) and Basal Metabolic Rate (BMR). This helps users understand their body’s needs and set realistic fitness goals.
  • Exercise List: Users can log their daily workouts, track their progress, and stay motivated by seeing their exercise history.
  • Motivational Quotes: Every day, users receive a new motivational quote designed to inspire and encourage them to stay focused on their fitness journey.
  • Eating List: The app includes an eating list feature, allowing users to plan their meals and track their daily intake, promoting a balanced diet.
  • Calendar Feature: Users can schedule their workouts and meal plans using the built-in calendar, ensuring consistency and organization in their routines.
  • Clear Local Storage: Users can reset their data with a simple "Clear All Local Storage" feature, helpful for troubleshooting or starting fresh.

Technology Used: This application is built using modern web technologies, including HTML, CSS, JavaScript, and various APIs to enhance functionality.

The Fitness Tracking App is designed to provide users with all the tools they need to stay on track with their fitness and health goals. From BMI tracking to meal planning and motivational support, it is a one-stop solution for fitness enthusiasts of all levels.

Memory Game (C# Windows Forms)

The Memory Game is a simple yet engaging game that challenges players to match tiles based on different difficulty levels. Built using C# Windows Forms, the game provides an interactive way for users to test their memory and concentration skills.

Game Concept: The game consists of various tiles that are initially hidden. The objective is to reveal and match pairs of tiles based on the selected difficulty level.

Key Features:

  • Easy Level: Players are presented with colored tiles. The goal is to match pairs of tiles with the same color. Players are given a limited number of tries, and if the counter reaches zero, the player loses.
  • Medium Level: This level uses numbered tiles instead of colors. The objective remains the same: match pairs of tiles with the same number within the given number of tries.
  • Hard Level: The hardest level involves matching photos. Players must match identical images, making the game more challenging as it requires sharper focus and a better memory.

Game Mechanics: Each level includes a try counter, which tracks the number of attempts the player has made. If the counter reaches zero before all pairs are matched, the player loses the game. The player must strategize and remember tile locations to succeed.

Technology Used: The game is developed using C# and Windows Forms, providing a simple yet effective environment for game development and interaction.

The Memory Game is designed to challenge and improve the player’s memory while offering a fun and rewarding experience with three difficulty levels that cater to different skill levels.
